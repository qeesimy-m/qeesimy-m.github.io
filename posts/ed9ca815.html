网页部署与自定义域名的配置

发表于2023-08-12|更新于2023-09-29|博客教程
|阅读量:

Vercel部署

Vercel是一个代码托管平台，它能够托管你的静态html界面，甚至能够托管你的node.js与Python服务端脚本，是不想买服务器的懒人的福音,它在全世界都有站点，能加速网站访问

  1. 首先需要一个Vercel账号，这里推荐用GitHub账户关联，这样你就可以在vercel中直接托管你的GitHub库中的项目了，实现开发部署一步到位
  2. 当你用你的Github账户关联并绑定手机号登录之后，点击右上角的Add New Project创建新的项目，之后导入选项那里选择Continue with Github，这时候应该能看到你Github账号的仓库，选择你部署成功的存储静态博客的仓库<username>.github.io右边的Import选项，表示你要导入该仓库
  3. 起一个只能有字母、数字或者或者连字符的项目名称，然后其他默认，点击Deploy，等待一分钟即可部署成功，部署成功后点击Continue to Dashboard跳转到控制面板，下图所示就是控制面板，看到就代表成功部署，但是还不能访问他给出的域，因为GFW最近把Vercel屏蔽了

自定义域名

首先要拥有域名，静态博客要花钱的地方也就域名，平均下来一月几块

域名获取与购买

建议注册.top域名，综合价钱较低

一般推荐的有以下几个（我选的硅云）
eu.org：白嫖域名，但是要等
硅云：有首年0元域名的活动，便宜
阿里云：未成年人也能用
腾讯云：仅限18岁以上实名认证用户使用（像我就用不了）
NameSilo：号称国外最便宜的域名注册服务商

域名解析

购买域名后打开域名管理面板，点进域名解析列表（DNS解析管理）

点击添加记录，需要添加两个的记录为

记录类型 主机记录 记录值
CNAME www 博客地址
CNAME @ 博客地址

博客地址：xxx.github.io（Github Page自带的地址，参考图片）

关于这部分的说明：

主机记录 解释
www 解析后的域名为www.zeroqsm.top
@ 直接解析主域名zeroqsm.top
* 泛解析，匹配其他所有域名*.zeroqsm.top
mail 将域名解析为mail.zeroqsm.top，通常用于解析邮箱服务器
二级域名 如abc.zeroqsm.top，填写abc
手机网站 如m.zeroqsm.top，填写m
显性URL 不支持泛解析（泛解析：将所有子域名解析到同一地址）
记录类型 解释
A 用来指定域名的 IPv4 地址（如 8.8.8.8），如果需要将域名指向一个 IP 地址，就需要添加 A 记录
CNAME 如果需要将域名指向另一个域名，再由另一个域名提供 IP 地址，就需要添加 CNAME 记录
MX 如果需要设置邮箱，让邮箱能收到邮件，就需要添加 MX 记录
TXT 在这里可以填写任何东西，长度限制 255。绝大多数的 TXT 记录是用来做 SPF 记录（反垃圾邮件）
NS 域名服务器记录，如果需要将子域名交给其他 DNS 服务商解析，就需要添加 NS 记录
AAAA 用来指定主机名（或域名）对应的 IPv6 地址（例如：ff06:0:0:0:0:0:0:c3）记录
SRV 记录了哪台计算机提供了哪个服务；格式为：服务的名字、点、协议的类型，例如：_xmpp-server_tcp
显性 URL 从一个地址 301 重定向到另一个地址的时候，就需要添加显性 URL 记录（注：DNSPod 目前只支持 301 重定向）
隐性 URL 类似于显性 URL，区别在于隐性 URL 不会改变地址栏的域名

Github 添加域名

打开你的github博客项目，点击settings，点击Pages，拉到下面Custom domain处，填上你自己的域名，保存

添加 CNAME 文件

这时你的博客资源目录下应该会出现一个CNAME文件，如果没有，就主动添加
注意没有任何后缀名，而且是cname的大写！（我被这个坑过呜呜）
添加文件[Blogroot]/source/CNAME
文件里面填写你的域名（加不加 www 都行）
比如要填写我的域名，文件里面就写：www.zeroqsm.top或者zeroqsm.top
经过以上操作，别人就可以通过www.zeroqsm.top、zeroqsm.top、qeesimy-m.github.io中的任意一个地址访问我的博客了

加不加www的区别：（省流，都行）

  • 如果你填写的是没有www的，比如zeroqsm.top，那么无论是访问https://www.zeroqsm.top还是https://zeroqsm.top，都会自动跳转到https://zeroqsm.top
  • 如果你填写的是带www的，比如www.zeroqsm.top，那么无论是访问https://www.zeroqsm.top还是https://zeroqsm.top，都会自动跳转到http://www.zeroqsm.top
